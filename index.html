Hacker News RSS feeds

hnrss provides custom, realtime RSS feeds for Hacker News.

Feeds

The following feeds are available:

  • Firehose — Every new post and comment as it arrives
  • Points — Posts and comments with more than N points
  • Activity — Posts with more than N comments
  • Self-posts — All "Ask HN" and "Show HN" posts, along with polls
  • Users — New posts and comments made by a given user
  • Threads — Each new comment made in a given thread
  • Searches — New posts and comments matching a given search term

Firehose

The "firehose" feeds contains all new posts and comments as they appear on Hacker News.

Points

Maybe you like the idea of the firehose, but want to slow it down just a bit. You can do that with the ?points=X filter. With this filter added on to a firehose feed, only posts or comments with more than X points will be included.

So, for example, here's a feed of posts with 200+ points and comments with 100+ points.

Caveat: Comments and the points filter

There is one thing you should know when using the points filter on comments: It takes five days for point values on comments to be accurately reported via the Algolia API. What this means is there is a five day delay between when a comment is posted and when it shows up in an RSS feed when using the points filter.

This is a known issue. My guess is the volatile nature of comment voting makes it more trouble than its worth to report a comment's realtime points value.

Note: The comments firehose itself doesn't have this limitation, its only when using the points filter.

Self-posts

If you're looking for RSS feeds for Ask HN, Show HN, or polls, we've got you covered.

Users

If you don't want to miss a post or comment by a given user, you can subscribe to that user's RSS feed.

The URLs for both posts and comments, just posts, and just comments are (respectively):

  • http://hnrss.org/user?id=USERNAME
  • http://hnrss.org/submitted?id=USERNAME
  • http://hnrss.org/threads?id=USERNAME

With USERNAME replaced with the username in question.

As a shortcut, when on a user's profile page you can replace "https://news.ycombinator.com/" with "http://hnrss.org/". This also works on a user's submitted and comments pages.

Threads

If there's some thread you're particularly interested in, you can subscribe its RSS containing new comments in that thread at http://hnrss.org/item?id=THREAD_ID.

THREAD_ID is the number after ?id= when viewing a post's comments page (Example). Or, as a shortcut, replace "https://news.ycombinator.com/" with "http://hnrss.org/" when on a comments page to get the RSS feed.

Searches

You can also filter based on search terms via the ?q=SEARCH_TERM query filter.
